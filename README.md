# gameBase
A base for creating platform independant 2d games. 

## Actual implementation
### Common
- physique constants
- object class 
### Control
- 6 positions joystick (left, right, up, down, pressed, center)
### Elements
- ball 
- platform
- playable ball
### Graphics
- abstract display
### Physics
- physics simulator with 1d elastic collision
- 2d vector
- rectangle bounding box

## Target platform
- QT
- STM32F4/7 discovery
